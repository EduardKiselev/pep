Описание проекта:

Планируется создать сайт для подбора рациона домашних животных. В работе будет использоваться база данных USDA (US Department of Agriculture) https://fdc.nal.usda.gov/api-guide.html в данных момент я выкачал часть данных в json файле и тестово гоняю на нем.

1) калькулятор необходимого питомцу рациона: по заданным параметрам питомца(тип, возраст, пол, беременна/не беременна) считается необходимое количество питательных веществ которые требуются животному. [это еще не сделано]
2) калькулятор рациона: по заданному рациону(граммы каких либо продуктов) считаем сколько питательных элементов в этом рационе [это сделано view функция index]
	через админку идет управление: а) необходимыми нутриентами, которые нужно учитывать
				       б) едой которая доступна для добавления в рацион
3) функция поиска наболее подходящего ингридиента по заданной целевой функции в нутриентах. [это не реализовано]
