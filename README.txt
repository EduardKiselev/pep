  Описание проекта
  

  Планируется создать сайт для подбора рациона домашних животных. 
  В работе будет использоваться база данных USDA (US Department of Agriculture) https://fdc.nal.usda.gov/api-guide.html. 
  На GitHub есть часть данных в фикстуре small_data.json
  Информация о необходимых нутриентах для определнных питомцев находится на странице ./rec-nutr-lvl
  Калькулятор необходимого питомцу рациона: по заданным параметрам питомца(тип, возраст, пол, беременна/не беременна) 
      считается необходимое количество питательных веществ которые требуются животному. страница ./calc
  Поиск продукта: по заданной целевой функции из нутриентов находится самый подходящий продукт 

  Питомцы и рационы сохраняются в соответсвующих таблицах
